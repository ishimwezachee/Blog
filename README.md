# Blog

The Blog app is an example of a blog website. It's a functional website that shows the list of posts and enable users to interact with it by adding comments and liking posts.

## Built With

- Ruby
- Ruby on Rails

## Getting Started

To get a local copy up and running follow these simple example steps.

git clone https://github.com/ishimwezachee/Blog.git

cd Blog

### Prerequisites

Ruby
Rails
PostgresSQL

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
```

### Usage

Start server with:

```
rails s
```

Open http://localhost:3000/ in your browser.


### Run tests

Install npm with:

```
npm i
```

Install rspec with:

```
bundle install
```

and

```
rails generate rspec:install
```

run the test with:

```
rspec spec
```
